Welcome to the world of
Charles Grealy!

I am an adult, cat dad, and billionaire. One of these, believe it or not, happens to be a lie. In May 2020 I graduated from the University of Houston with a bachelor’s in Mathematics. All through primary school math was both my favorite and strongest subject so it felt fitting to study it in college.

Before attending college, I decided my career plan would be to become a programmer. I came to this decision after discovering certain programming fields require a strong math background. I researched what math graduates were doing in their career and quite a lot had successful careers as a programmer. Including my dad, who was a programmer for 40 years and coincidentally a math major!

In order to gain programming experience, I chose to minor in Computer Science. Well, after the first introductory to programming course I would quickly learn that computer science would be a subject that feels rather natural. I felt less inclined to take class notes as the material spoke to me fluently. The abundance of material (ex: books, articles, StackOverflow, and FreeCodeCamp) available is absolutely shocking. Programming sure does inject serotinin into my veins.

I’m now applying to various entry level positions and I am eager to start my career in programming. Once I obtain employment, my goal is to pursue a masters and a doctorate in computer science. If I could offer you only one tip, wear sunscreen.

Personal Website

A quarantine project was to create my own website from the ground up. I'll say it's the most fun I have yet to date! The goal in the beginning was to only develop a blog, but that later changed when I wanted to expand on the challenge. First acclimated myself with django (Python), where I learned about about url mapping, how to create views, and configure a database powered by PostgreSQL. Then I would create a layout from HTML/CSS and implement JavaScript to create pleasent features such as smooth scrolling and a drop down menu in the blog posts.

GitHub

Stock Portfolio Chatbot

Another quarantine project based upon an interest of mine, the stock market. Yep, I am completely invested in the stock market - pun intended. With Flask (Python) and the help of the Twilio API came a chat bot used in WhatsApp. Since I became sort of acclimated with django during development, Flask felt more natural - in fact maybe quite easy to setup. A challenge I faced was learning how to create GET and POST requests for user interactivity. The other challenge, more like a complete headache, was configuring the PostgreSQL database. I faced issues getting the database connected to Heroku when pushing into live production.

GitHub

Python Interpreter in C++

One of the final projects for a class project was to create a Python interpreter in C++. One of my favorite projects during my undergrad. Paired with a great partner, together we would be victorious. The objective was to take a Python (.py) file as input, parse variables and lists, and output list contents. So it's as if Python ran the program, but it's all C++ behind the scenes.
